## 🎮 Sobre o Projeto

**Nerd Tech Hub** é um projeto que combina cultura nerd com tecnologia.

Ele possui dois módulos principais:

⚔️ **Anime Battle Simulator (C)**  
Simula batalhas entre personagens de anime utilizando estruturas em C.

🤖 **AI Project Risk Predictor (Python)**  
Um pequeno sistema de Machine Learning que prevê o risco de um projeto de TI.

Este projeto foi criado para demonstrar habilidades em:

- Programação em **C**
- **Python**
- **Machine Learning**
- Estruturação de projetos
- CLI Applications
